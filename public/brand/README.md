# Quiroz Automotriz — Brand Assets

Coloca aquí tus archivos reales con los nombres indicados. La landing los tomará automáticamente.

| Archivo | Tamaño recomendado | Uso |
|---|---|---|
| `logo.svg` o `logo.png` | ~200x60 | Logo principal (navbar) |
| `logo-dark.svg` | ~200x60 | Logo para fondos oscuros |
| `favicon.ico` | 32x32 / 16x16 | Ícono de la pestaña del navegador |
| `og-image.png` | 1200x630 | Vista previa en redes sociales |
| `hero-bg.jpg` | 1600x900 | Imagen principal del hero |
| `about-bg.jpg` | 1200x800 | Imagen de la sección Nosotros |
| `service-1.jpg` … `service-6.jpg` | 1200x900 | Imágenes de las tarjetas de servicios |
| `testimonial-1.jpg` … `testimonial-3.jpg` | 200x200 | Fotos de clientes (avatares) |

Mientras no subas archivos reales, se usan imágenes de Unsplash como placeholder.
